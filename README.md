# 🚀 SkillBridge: Engineering a Trust-Driven Peer Skill Economy
**🏆 Top 100 Finalist (out of 900+ teams) at Elite Hackathon*

SkillBridge is a hyperlocal peer-to-peer skill exchange platform where knowledge is the currency. It connects students within a community to trade skills—like teaching Python in exchange for learning UI Design—without any monetary transaction.

## 💡 The Problem
Students often lack the funds for expensive online courses but possess valuable skills themselves. Current platforms like Udemy or LinkedIn are either pay-to-play or purely social, leaving a gap for structured, trust-based mutual collaboration.

## ✨ Key Features
- **Mutual Skill Symmetry Engine:** An intelligent matching algorithm that identifies "exchange loops" between users (User A teaches X, User B teaches Y).
- **Reputation Score System:** A trust-based rating system to ensure quality and accountability in peer learning.
- **Hyperlocal Ecosystem:** Optimized for university campuses to facilitate high-trust, face-to-face or local digital collaboration.


## 🚀 How to Run Locally
1. Clone the repo: `git clone https://github.com`
2. Install dependencies: `npm install` (or your specific command)
3. Start the app: `npm start`
